# Research Source Register - Module 3

## Overview

This document tracks all research sources used in Module 3: AI-Robot Brain (NVIDIA Isaac). Each source is validated and properly cited according to APA format with DOI, link, summary, and relevance note.

## Source Tracking Format

Each entry follows the format:
- **Source ID**: Unique identifier
- **DOI**: Digital Object Identifier
- **Link**: Direct URL to source
- **Summary**: Brief description of content
- **Relevance Note**: Why this source is relevant to the module
