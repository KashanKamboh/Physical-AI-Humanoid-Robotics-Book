Course Roadmap & Timeline

Academic Calendar Overview​

This roadmap outlines the 12-16 week course structure for the Physical AI & Humanoid Robotics curriculum.

Module Timeline​

Module 1: Robotic Nervous System (ROS 2) - Weeks 1-4​

Learning Objectives:

  • Understand ROS 2 architecture and communication patterns
  • Create and manage ROS nodes, topics, and services
  • Implement basic robot control through ROS

Weekly Breakdown:

  • Week 1: Introduction to ROS 2, workspace setup, basic concepts
  • Week 2: Nodes, publishers, subscribers, message passing
  • Week 3: Services, actions, launch files, parameter management
  • Week 4: URDF modeling, robot description, basic control

Assessment: Basic ROS node that publishes sensor data and responds to commands

Module 2: Digital Twin (Gazebo & Unity) - Weeks 5-8​

Learning Objectives:

  • Create and configure simulation environments
  • Integrate robot models with physics engines
  • Test control algorithms in virtual environments

Weekly Breakdown:

  • Week 5: Gazebo setup, world creation, physics parameters
  • Week 6: Robot spawning, sensor integration, collision systems
  • Week 7: Unity robotics tools, humanoid rigs, motion blending
  • Week 8: Cross-platform simulation, validation techniques

Assessment: Complete robot simulation with multiple sensors functioning

Module 3: AI-Robot Brain (NVIDIA Isaac) - Weeks 9-12​

Learning Objectives:

  • Implement perception pipelines for robot awareness
  • Navigate environments using SLAM and path planning
  • Execute manipulation tasks with precision

Weekly Breakdown:

  • Week 9: Isaac Sim setup, USD scene format, camera feeds
  • Week 10: Mapping, localization, path execution with Nav2
  • Week 11: Perception pipelines, detection, segmentation
  • Week 12: Manipulation planning, grasp generation

Assessment: Autonomous navigation task with obstacle avoidance

Module 4: Vision-Language-Action (VLA) - Weeks 13-16​

Learning Objectives:

  • Integrate voice recognition with robot action planning
  • Implement cognitive task planning for complex behaviors
  • Execute end-to-end voice-commanded robot operations

Weekly Breakdown:

  • Week 13: Voice-to-action agents, Whisper integration
  • Week 14: Task breakdown algorithms, planning systems
  • Week 15: Human-robot interaction patterns, safety considerations
  • Week 16: Capstone integration and demonstration

Assessment: Complete VLA system responding to natural language commands

Milestone Checkpoints​

Module Completion Criteria​

Each module must demonstrate:

  • ✅ Working implementation of core concepts
  • ✅ Validated through simulation or hardware testing
  • ✅ Proper documentation and code quality
  • ✅ Peer review and instructor approval

Capstone Timeline​

  • Week 14: Capstone project proposal and planning
  • Week 15: Integration of all modules into unified system
  • Week 16: Final demonstration and evaluation

Resource Allocation​

Time Investment​

  • Class Time: 3 hours per week (2 lectures + 1 lab)
  • Lab Time: 2-3 hours per week for assignments
  • Project Work: 4-6 hours per week for module projects

Assessment Distribution​

  • Module Projects: 60% (15% each module)
  • Weekly Assignments: 20%
  • Capstone Project: 20%
